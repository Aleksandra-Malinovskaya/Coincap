# Coincap

SPA-приложение для мониторинга курсов криптовалют в реальном времени и управления персональным инвестиционным портфелем.

# Ключевые функции

Мониторинг рынка: Загрузка и отображение ТОП-100 популярных криптовалют с актуальными данными о цене, рыночной капитализации и изменениях за 24 часа

Интерактивные графики: Визуализация истории изменения цены за последние 10 часов с использованием Recharts. Настроено кастомное форматирование осей (только часы) для лучшей читаемости.

Управление портфелем:
Возможность добавления валют с проверкой корректности ввода (только положительные числа, автоматическая замена запятой на точку).
Расчет общей стоимости активов, абсолютной прибыли/убытка и процентного изменения портфеля относительно цен покупки.
Синхронизация данных с localStorage для сохранения состояния между сессиями.
Удобная навигация: Реализована пагинация (по 10 элементов на страницу) и детальные страницы для каждой монеты с использованием React Router.

# Технологический стек

React: Использование хуков useState, useMemo, useEffect и useParams для управления логикой компонентов.
TanStack Query (v5): Эффективное управление серверным состоянием: кэширование данных, автоматическое обновление и обработка состояний загрузки (isPending) и ошибок
Material UI (MUI): Современный интерфейс с использованием готовых компонентов (Table, Modal, Box, TextField, IconButton).
Coincap API: Использование внешнего API для получения актуальных рыночных данных.
Recharts: Библиотека для построения адаптивных графиков

# Реализация и качество кода

Синхронизация данных: Приложение разделяет "клиентское состояние" (состав портфеля в localStorage) и "серверное состояние" (актуальные цены из API), объединяя их для точных расчетов прямо в интерфейсе

Валидация: Реализована строгая фильтрация пользовательского ввода в формах покупки через регулярные выражения.
Производительность: Использование useMemo для тяжелых вычислений (например, расчет итоговой суммы портфеля) и оптимизация ResponsiveContainer для корректного отображения графиков на любых экранах.

# Как запустить проект локально

Клонируйте репозиторий.
Установите зависимости: npm install.
Создайте файл .env и добавьте необходимый ключ.
Запустите проект: npm run dev.
